Primer Corte

En este corte la tematica se trata del funcionamiento del sistema operativo y el uso de comandos en CMD y PowerShell.

CMD PowerShell

CMD - Mantenimiento Windows 11

Procesador y sistema

systeminfo → Información detallada del sistema.

perfmon /report → Informe completo de rendimiento.

Disco duro

chkdsk C: /f /r → Revisa y repara errores.

fsutil dirty query C: → Verifica si el disco necesita reparación.

Memoria RAM

mdsched.exe → Diagnóstico de memoria.

systeminfo → Muestra memoria instalada.

Archivos del sistema

sfc /scannow → Repara archivos dañados.

DISM /Online /Cleanup-Image /RestoreHealth → Repara imagen de Windows.

PowerShell - Mantenimiento Windows 11

Procesador y sistema

Get-WmiObject Win32_Processor → Información del CPU.

Get-ComputerInfo → Datos completos del sistema.

Get-Process | Sort-Object CPU → Procesos que más consumen CPU.

Disco duro

Get-PhysicalDisk → Estado de discos físicos.

Get-WmiObject Win32_LogicalDisk → Espacio libre.

Repair-Volume -DriveLetter C -Scan → Escanea disco.

Repair-Volume -DriveLetter C -OfflineScanAndFix → Repara errores.

Memoria RAM

Get-WmiObject Win32_PhysicalMemory → Información RAM.

Get-Counter '\Memory\Available MBytes' → Memoria disponible.

Batería

powercfg /batteryreport → Reporte de batería.

Get-WmiObject Win32_Battery → Estado básico batería.

Procesos y rendimiento

Get-Process | Sort-Object CPU → Procesos por CPU.

Get-Process | Sort-Object PM → Procesos por memoria.

Get-Counter '\Processor(_Total)\% Processor Time' → Uso total CPU.

Programas instalados

Get-WmiObject Win32_Product → Lista programas.

Get-Package → Paquetes instalados.

Get-WindowsUpdateLog → Historial actualizaciones.
